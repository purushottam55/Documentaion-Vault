Hotfix / Interim Patch
- Purpose: Emergency fix for critical issues before official advisory release.
- Advisory Code: None (delivered directly by Red Hat Support).

- Example: Custom kernel RPM provided for urgent production issue.
- Scope: Temporary fix, later included in official RHSA/RHBA/RHEA.
- Command Examples:
    rpm -Uvh custom-hotfix.rpm
    yum localinstall custom-hotfix.rpm
