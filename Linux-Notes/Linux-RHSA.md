- RHSA is Red Hat’s official advisory notice.
- Example: RHSA-2025:1234 → Security update for OpenSSL.
- Issued by Red Hat Product Security team.
- Provides: 
   * Which RHEL versions are affected
   * Which packages are updated
   * Fix details (and the CVEs included)
- Vendor-specific, actionable patch info.
- Tells you *how Red Hat fixed the flaw*.
