
# Title  : On which os the OpenShift runs.  

-----
### **Primary Supported Host OS**  
- **Red Hat Enterprise Linux CoreOS (RHCOS)** (default for control plane)  
- **Red Hat Enterprise Linux (RHEL)** (worker nodes)  

### **Other Supported Platforms**  
- **Fedora CoreOS** (for OKD, the community version)  
- **Windows Server** (worker nodes only, for Windows containers)  

### **Cloud/Managed Options**  
- **AWS, Azure, GCP, IBM Cloud** (via OpenShift Dedicated, ROSA, ARO, etc.)  
- **VMware vSphere, Bare Metal, OpenStack** (RHCOS/RHEL-based)  

*(OpenShift does not support generic Linux distros like Ubuntu or Debian for host nodes.)*
